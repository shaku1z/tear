## Honest Certification Boundary

- The visible autonomous lifecycle is Class B engineering evidence. It is not relabeled as Class C.
- Normal Adventure has no Class C certification claim until the separate statistical threshold is met.
- Population plausibility remains explicitly provisional until consented samples exist.
- Cloud deployment still requires real environment-owned D1/R2 IDs, authentication, signing secrets, and the trusted historical verifier service. Local gameplay and recording do not depend on them.
